# Deskripsi Tugas
Tugas ini bertujuan melakukan proses pengintaian (Reconnaissance) secara pasif dan aktif terhadap sebuah target, kemudian mendokumentasikannya dalam bentuk laporan, screenshot, dan video.


# 1. PASSIVE RECON
Tahapan:
- Mengidentifikasi domain
- Mengumpulkan subdomain menggunakan crt.sh
- Menemukan format email resmi
- Menganalisis teknologi website menggunakan Wappalyzer
- Mencari informasi sensitif via Google Dorks


# 2. ACTIVE RECON
Tahapan:
- Ping target
- Mengambil IP interface (ifconfig)
- Host discovery (nmap -sn)
- Port scanning (nmap -sS)
- Service version detection (nmap -sV)
- OS detection


# 3. SKEMA TUGAS
Daftar langkah pengerjaan:
1. Menentukan target
2. Melakukan Passive Recon
3. Melakukan Active Recon
4. Mengambil semua screenshot
5. Membuat laporan
6. Merekam video
7. Upload ke YouTube
8. Upload file ke GitHub
9. Submit ke Classroom
